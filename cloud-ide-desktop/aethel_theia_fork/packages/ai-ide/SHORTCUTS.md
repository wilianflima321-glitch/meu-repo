## Tips

### Chord Shortcuts
Some shortcuts use "chord" sequences (two key combinations in sequence):
- `Ctrl+K Ctrl+S`: First press `Ctrl+K`, then press `Ctrl+S`
- Release all keys between the two combinations

### Custom Shortcuts
You can customize any shortcut:
1. Open Command Palette (`Ctrl+Shift+P`)
2. Type "Preferences: Open Keyboard Shortcuts"
3. Search for the command
4. Click the pencil icon to edit
5. Press your desired key combination

### Conflicts
If a shortcut doesn't work:
- Check for conflicts with OS shortcuts
- Check for conflicts with browser shortcuts
- Use Command Palette as alternative

### Learning Shortcuts
- Hover over buttons to see shortcuts in tooltips
- Use Command Palette to discover commands
- Check "Help → Keyboard Shortcuts Reference"
