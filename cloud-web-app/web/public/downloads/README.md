# Downloads

Place release artifacts for Aethel IDE here so they are served at:
- /downloads/aethel-ide-windows.exe
- /downloads/aethel-ide-mac.dmg
- /downloads/aethel-ide-linux.tar.gz

Include the following verification files:
- checksums.txt (SHA-256)
- PGP_KEYS.txt (public keys used to sign releases)
- RELEASE_NOTES.md (changelog for the current release)

Note: This folder is served publicly by Next.js. Avoid placing sensitive files here.
