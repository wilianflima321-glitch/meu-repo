## 1. As 3 Áreas de Domínio

Para superar concorrentes como Cursor, VS Code, Unreal, Manus e GenPark, o Aethel Engine é dividido em três camadas de experiência progressiva.

| Área | Nome | Foco | Diferencial de Superação |
| :--- | :--- | :--- | :--- |
| **Área 1** | **The Gateway (Web)** | Onboarding & Conversão | Experiência "Instant On" com Magic Box. |
| **Área 2** | **The Nexus (Home)** | Colaboração & Live Preview | Chat Multimodal estilo Gemini + Canvas Interativo. |
| **Área 3** | **The Forge (IDE)** | Engenharia de Alta Performance | Superior ao VS Code em IA; Superior à Unreal em Web Runtime. |

---

## 2. Área 2: The Nexus (A "Home Page" Evoluída)

O Nexus não é um dashboard estático. É um **Ambiente de Orquestração Viva**.

### 2.1. Live Preview Interativo (Canvas 2.0)
- **Visualização de Pensamento:** O usuário não apenas vê o resultado, mas assiste a IA "pintando" o código e os assets no canvas.
- **Interatividade Total:** Clique em qualquer elemento no 3D ou UI para abrir um mini-chat contextualizado (estilo Magic Wand).
- **Streaming de Execução:** Integração com WebContainers para rodar código real instantaneamente com hot-reload visível.

### 2.2. Chat Multimodal (Estilo Gemini/Canvas)
- **Memória de Longo Prazo:** A IA conhece todo o repositório e documentos canônicos.
- **Modo "Observador":** A IA sugere melhorias proativamente enquanto o usuário navega no Live Preview.
- **Squad de Agentes:** Alternância entre perfis (Arquiteto, Designer, Dev, QA) dentro da mesma thread.

---

## 3. Área 3: The Forge (A IDE de Próxima Geração)

Superar o VS Code requer ir além de um editor de texto com plugins.

### 3.1. IA de Primeira Classe
- **Deep Context:** Diferente do Copilot, a IA do Aethel opera sobre a "Reality Matrix" do projeto, entendendo intenções de negócio, não apenas sintaxe.
- **Autocorreção de Interface:** A IDE detecta desvios do `AETHEL_DESIGN_MANIFESTO` e sugere correções visuais automáticas.

### 3.2. Superação de Limitações Técnicas
- **Hybrid Rendering:** Uso de WebGL/WebGPU local para interface e Pixel Streaming para visualizações AAA (Unreal Engine na nuvem).
- **Arquitetura "Lean":** Otimizada para rodar em hardware modesto através de offloading de processamento pesado para a nuvem Aethel.
