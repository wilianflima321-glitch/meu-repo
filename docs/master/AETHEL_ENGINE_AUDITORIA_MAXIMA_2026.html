5. Módulo Games

Status: L2 experimental

Existe: WebGPU canvas, asset preview, WASM engine, scene graphs.

Falta: Meshy API, Rapier.js, XState, ElevenLabs, sprite gen, export Unity/Godot/Unreal, replay QA, tilemap editor.

Roadmap L3: integrar Meshy API, physics Rapier, state machine XState, export GLB/FBX, QA replay loop.

6. Módulo Films

Status: L2 experimental

Existe: Director Mode, shot timeline, workflow orchestration.

Falta: geração de vídeo real (Kling/Runway/Veo), audio-to-video, continuity, Elements, Retake, storyboard pipeline, NLE, export profissional.

Roadmap L3: integrar APIs de vídeo, continuity store, retake segmentado, export MP4.

7. Módulo Apps

Status: L3 beta

Forças: multi-agent, anti-fake-success, RBAC, CI/CD, dependency guard.

Gaps: @-mentions completas, RAG semântico, preview sandbox real, deploy one-click, Figma import, diff viewer, billing completo, colaboração sem stress test.

Caminho para L4: RAG persistente, HMR sandbox, deploy one-click, evidência de production runs.

8. Módulo Research Agent

Comparação: Perplexity Deep Research (67.15% DRACO, 780M queries/mês, citações inline).

Gaps: sem citações inline, sem web real-time, sem métricas públicas.

Diferencial único: research → code → deploy em loop único. Deve ser explicitado na UI.

9. Infraestrutura e Backend

Stack: Next.js 14, Prisma, AWS, Node WebSockets.

Sandbox: runtimeUrl parcial, sem E2B/WebContainers real.

Comparações: Bolt (WebContainers), Replit (VMs cloud), Codespaces ($0.18/hr).

Recomendação: E2B ($0.05/hr), Railway/Render para deploy, Upstash Redis/rate-limit.

10. Sistema de Pagamentos

Status: Parcial Stripe sem checkout completo.

Falta: pricing, planos, trial, upgrade self-service, invoices, webhooks completos.

Sem billing = zero receita recorrente.

11. Marketing e Go-to-Market

Status: Zero landing pública, pricing, social proof e canais.

Necessário: hero video, testimonials, pricing tiers, SEO pillar pages, Product Hunt, Twitter/X, YouTube, Discord.

Copy sugerida: “O único IDE com múltiplos agentes de IA que realmente dizem quando falham.”

12. Onboarding e First Value

Estado: rail com milestones e telemetria parcial, SLO < 90s não provado.

Fluxo ideal: signup → demo projeto → AI → preview → convida colaborador.

Targets: first action < 30s, first preview < 90s, first deploy < 5min.

13. Mobile e Responsividade

Estado: desktop-first. Mobile crítico para landing, onboarding e Nexus Chat view-only.

Comparação: Linear (mobile excelente), Replit (mobile limitado).

14. Acessibilidade

WCAG gate operacional, mas sem evidência completa de contraste.

Falta: contraste 4.5:1, skip links, focus trapping, navegação por teclado total.

15. Design System e Tokens

Estado: legacy tokens zerados, design system parcial.

Oportunidades: motion tokens, spacing scale 4/8/12/16/24/32/48/64, typography consistente, keyboard-first.

16. Segurança e Compliance

Existe: RBAC, security headers, audit trail, rate limits.

Gaps: SOC 2, GDPR docs, penetration test público, bug bounty.
